Release History¶

v0.0.5 (2021-08-28)¶

This release includes a variety of bug fixes and JOSS review process. Concerning bug fixes an argument swap in the copy_no_deid function was corrected, plotting functions were adapted (wrt T2w defacing and main plotting function) and mridefacer problems solved. Regarding the JOSS review process, multiple sections of the documentation were extended and several typos were fixed. A new section displaying an usage example was added to help interested users and allow software testing. As this uses an open non-deidentified dataset, the example data was removed. Additionally, the Singularity image was dropped due to recent developments around Singularity hub and adapted to suggest obtaining a Singularity image through a direct download from Dockerhub.

v0.0.4 (2021-01-16)¶

This version adds bidsonym to PyPi.

v0.0.3 (2021-01-16)¶

This release mainly consists of various bug fixes and minor new features. Bug fixes mainly addressed problems related to how sessions and corresponding were handled. New features in particular refer to reorganizing the outcomes into data (and if present session) specific directories and stable graphics generation.
